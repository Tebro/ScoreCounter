Demo

Requirements

  • Ruby 1.9+
  • Node.js
  • compass: gem install compass
  • bower: npm install bower -g
  • foundation: gem install foundation

Quickstart

Once you have all the dependencies installed run the following.

git clone https://github.com/Tebro/ScoreCounter.git
cd ScoreCounter
foundation update
compass compile
